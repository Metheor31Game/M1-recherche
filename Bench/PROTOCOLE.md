Remarques générales :
    1. Les algorithmes à tester sont : Robinson, Martelli-Montanari et le Discrimination Tree;
    2. Les structures de données à tester sont : Listes, Ensembles et le Discrimination Tree;
    3. D'après les points 1 et 2, Discrimination Tree est un cas "à part";
    4. D'après le point 3 il faudra donc tester : 
            - Robinson sur des listes;
            - Robinson sur des ensembles;
            - Martelli-Montanari sur des listes;
            - Martelli-Montanari sur des ensembles;
            - Discrimination Tree;
    5. Ces tests doivent être fait sur la/les même banque/s de données;
    6. Les résultats de ces différents tests doivent être les mêmes;
    7. Pour tester facilement que ces différents résultats soient les mêmes, ils doivent donc avoir le même format;
